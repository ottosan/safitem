#Safitem

避難所で不足している品物一覧を表示し，避難所と物資支援をしてくれる企業，個人を結ぶサービス

###ビュー構成

####index.erb

トップページ．とりあえず今はリンクを作っておくだけ．あとログイン，ログアウト画面とか．
